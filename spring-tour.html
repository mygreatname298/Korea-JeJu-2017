韓國濟州、本島南部春遊 – 寫在出發

我們今天 (2017年3月22日)深夜便出發往韓國濟州和韓國本島南部旅遊, 2017年4月13日回程返港。因近幾個月都忙於家中的事情, 而也很遲才決定這次旅程, 所以準備的時間比較倉促, 雖然路線已經盡量簡單, 但很多交通在出發前還未弄清楚, 部份路線還未能決定, 只希望一切順利。

我們這次韓國濟州、本島南部春遊的粗略旅遊路線:

[濟州島] 香港 → 濟州 → 中文 → 西歸浦 → 表善 → 城山 →濟州市 →
濟州乘船往木浦 →
[韓國本島南部] 木浦 → 莞島 → 青山島 → 海南 → 順天 → 南海 → 釜山 → 香港
